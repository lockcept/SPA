# SPA (Score-based Preference Augmentation)

## Installation

install python requirements

```bash
pip install -r requirements.txt
```

install d4rl

```bash
git clone https://github.com/Farama-Foundation/d4rl.git
cd d4rl
pip install -e .
```

## Run

```bash
python main.py --help
python main.py -e box-close-v2 -f 1
```
