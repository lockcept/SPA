# Trajectory Cutter

## Installation

install python requirements

```bash
pip install -r requirements.txt
```

install d4rl

```bash
git clone https://github.com/Farama-Foundation/d4rl.git
cd d4rl
pip install -e .
```

install OfflineRL-Kit

```bash
cd OfflineRL-Kit
pip install -e .
```

## Run

```bash
python main.py --help
python main.py -e maze2d-medium-dense-v1 -f 1
```
